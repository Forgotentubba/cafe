# ☕ Café Club

Um site moderno e responsivo para uma cafeteria, desenvolvido com **HTML e CSS puro**.
O projeto apresenta um layout elegante com menu, avaliações de clientes e localização.

---

## 🚀 Demonstração

📍 Projeto simula uma landing page de cafeteria com:

* Página inicial com destaque
* Seção "Sobre"
* Cardápio (menu)
* Avaliações de clientes
* Localização com mapa

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts
* Icons8 (ícones)

---

## 📂 Estrutura do projeto

```
📁 projeto
 ├── index.html
 ├── style.css
 └── 📁 img
```

## 🎨 Funcionalidades

✔️ Layout moderno e responsivo
✔️ Menu com produtos e preços
✔️ Seção de avaliações de clientes
✔️ Integração com Google Maps
✔️ Navegação por âncoras
